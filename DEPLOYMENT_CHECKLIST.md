## 📋 部署步驟

### 1. 部署前檢查
```bash
# 檢查代碼質量
npm run lint
npm run type-check

# 運行測試
npm run test

# 構建應用
npm run build

# 檢查構建輸出
ls -lh dist/
```

### 2. 數據庫遷移
```bash
# 運行遷移
npm run db:migrate

# 驗證遷移
npm run db:verify
```

### 3. 環境配置
```bash
# 設置環境變量
export NODE_ENV=production
export API_URL=https://api.simeng.tw
export DATABASE_URL=postgresql://...
```

### 4. 部署應用
```bash
# 部署到生產環境
npm run deploy

# 驗證部署
curl https://www.simeng.tw/health
```

### 5. 部署後驗證
```bash
# 檢查應用狀態
npm run health-check

# 運行煙霧測試
npm run smoke-test

# 檢查性能指標
npm run performance-check
```

## 🚨 回滾計劃

### 回滾條件
- 應用無法啟動
- 關鍵功能無法使用
- 數據庫連接失敗
- API 響應時間 > 5s
- 錯誤率 > 5%

### 回滾步驟
```bash
# 1. 停止當前版本
npm run stop

# 2. 恢復上一個版本
npm run rollback

# 3. 驗證恢復
npm run health-check
```
